Modes:

  • ANGRY: Only uppercase code works
  • CALM: Only lowercase code works
  • CHAOS: All operations double their effect
  • NORMAL: Regular behavior

Commands:

  • MODE <mode> - Set the interpreter mode
  • SET X=10 - Set variable X to value 10
  • ADD X 5 - Add 5 to variable X
  • SUB X 5 - Subtract 5 from variable X
  • MUL X 2 - Multiply variable X by 2
  • DIV X 2 - Divide variable X by 2
  • ADDV X Y - Add the value of Y to X
  • SHOUT X - Print the value of X
  • whisper x - Print the value of X (in CALM mode)
